# Changelog

Все значимые изменения в проекте будут документированы в этом файле.

## [1.0.0] - 2025-01-10

### Добавлено

#### Основные функции
- Парсинг каналов (комментарии)
- Парсинг чатов (группы)
- Временные фильтры активности (неделя, месяц, 3 месяца)
- Генерация Excel отчетов с тремя листами (Users, Admins, Raw Data, Statistics)
- Генерация TXT отчетов со списком юзернеймов

#### Система аккаунтов
- Добавление пользовательских Telegram аккаунтов
- Поддержка 2FA авторизации
- Системная сессия для публичных ресурсов
- Безопасное хранение сессий

#### Система лимитов
- 10 бесплатных парсингов для новых пользователей
- Премиум подписка для безлимитного доступа
- Счетчик использованных парсингов

#### Админ-панель
- Выдача премиум статуса
- Сброс лимитов пользователей
- Статистика бота

#### Безопасность
- Случайные задержки между запросами (анти-детект)
- Обработка FloodWaitError
- Скрытие ADMIN_ID из отчетов
- Локальное хранение сессий

#### Интерфейс
- Интуитивное меню с инлайн-кнопками
- Подробные инструкции и помощь
- Прогресс-бар при парсинге
- Многоязычная поддержка (Русский)

#### Техническое
- База данных SQLite с тремя таблицами
- Асинхронная архитектура (asyncio)
- FSM для управления состояниями
- Логирование в консоль и файл
- Обработка ошибок

### Технологии

- Python 3.11+
- Aiogram 3.15.0
- Telethon 1.37.0
- Pandas 2.2.3
- OpenPyXL 3.1.5
- aiosqlite 0.20.0

### Файлы

- `config.py` - Конфигурация проекта
- `database.py` - Управление БД
- `keyboards.py` - Клавиатуры бота
- `main.py` - Точка входа
- `services/telethon_core.py` - Ядро парсинга
- `handlers/user_handlers.py` - Обработчики пользователей
- `handlers/admin_handlers.py` - Админ-функции
- `utils/excel_generator.py` - Генератор отчетов

### Документация

- `README.md` - Основная документация
- `SETUP.md` - Пошаговая установка
- `CHANGELOG.md` - История изменений
- `run.bat` - Запуск для Windows
- `run.sh` - Запуск для Linux/Mac
